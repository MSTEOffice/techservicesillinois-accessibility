Responsive Menu Example

This example showcases a menu found in the footer of the Adobe website. There were accessibility issues with this menu that would prevent screen reader users from successfully interacting with the menu. On this page is a modified example that corrects the accessibility issues while leaving the user interaction very similar to the original.

Note: This example should not be taken as a general indication of the accessibility of Adobe's products or services.
