Technology Services Accessibility Examples

Welcome to the Technology Services accessibility examples website. This site showcases examples of how to implement common elements of webpages and user interfaces accessibly.

On this site, you will find working examples of:

  • ARIA widgets and examples of implementing ARIA in webpages
  • Technology Services Bootstrap accessibility shim
  • Common areas where accessibility issues arise
  • Tests for accessibility support of various user agents
  • Miscellaneous examples that arise out of consultation with developers
